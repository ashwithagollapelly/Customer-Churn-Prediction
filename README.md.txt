# Customer Churn Prediction

## Project Overview
This project predicts whether a telecom customer is likely to churn using Machine Learning. The objective is to help telecom companies identify customers who are at risk of leaving and improve customer retention.

## Business Problem
Customer churn is a major challenge for telecom companies. Predicting which customers are likely to leave helps businesses take proactive retention measures.

## Dataset
- Dataset: Telco Customer Churn
- Number of Records: 7043
- Features: Customer demographics, services, billing details, contract information, and churn status.
